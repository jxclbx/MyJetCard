# PlaneHub 开发记录

---

## 2026年2月 — 初始构建

**2026-02-xx**
- 画廊页改用小缩略图（`image_sm`）展示，流量减少约 80%
- `Photo` 模型新增 `image_sm`（300 px）和 `image_lg`（1500 px）字段，保存时由 Pillow 自动生成，含锐化处理
- 照片详情页默认加载 `image_lg`，点击主图全屏显示原图；修复全屏放大时左侧被裁切的问题
- 上传约束：最大 3 MB，仅限 JPG/PNG，前后端双重校验
- 航司、机型、细分机型输入框新增自动补全（Autocomplete），实时从数据库拉取历史值，支持键盘导航
- 注册号搜索忽略横杠（`B6789` 可匹配 `B-6789`），机型改为模糊匹配
- 全局关键词搜索（`q=`），一次匹配注册号、机型、航司、机场、备注五个字段
- 导航栏搜索框统一绑定跳转逻辑，Home / 详情页 / 统计页均可直接跳转画廊
- 默认排序从"拍摄日期"改为"上传顺序（ID 倒序）"
- 地图底图从 Esri → OpenStreetMap → 高德极简风格（`style=7`），限制最大缩放级别，国内稳定可用
- 机场标记悬停浮窗、照片详情页机场字段、统计页排行榜和饼图，均改为显示机场全名（IATA 代码 → 名称映射，覆盖 9000+ 机场）
- 管理列表页加入缩略图预览；待审核队列优化布局与操作按钮
- 编辑照片页移除图片替换入口，仅保留元数据编辑
- API 筛选参数统一用循环处理，分页改用 Django 内置 `Paginator`，输出格式标准化
- 静态文件加入版本号参数（`?v=x.x`）用于缓存破坏
- 新建 `base.css`，统一全站 Inter 字体、抗锯齿和通用组件样式（`.btn`、`.card`、`.badge` 等）；manage 页面全部改引用 `base.css`，移除各自内联字体代码
- ECharts 图表注册 Inter 主题，统一字体渲染
- 修复访问别人主页时头像菜单仍弹出管理选项的问题
- 修复导航栏搜索框因变量未定义导致完全失效的问题

## 2026年3月 — 功能迭代

**2026-03-02**
- 梳理所有需要 staff/admin 权限的入口，完善权限保护
- 照片标签新增：彩绘（Special Livery）、货机（Cargo）、公务机（Bizjet）；画廊筛选支持这些标签
- 提交表单提交后禁用按钮，防止重复点击导致重复上传

**2026-03-06**
- 首页航司饼图显示数量从 6 改为 10

**2026-03-07**
- 移动端顶部导航栏高度压缩，视觉更紧凑
- 照片卡片日期显示改为上下两行布局（年份 / 月日分行）

**2026-03-09**
- 统计页排行榜条目可点击，直接跳转到对应筛选条件的画廊页
- 搜索和筛选支持细分机型（`sub_model`）字段
- 上传流程改为先进入待审核队列（`PendingPhoto`），staff 审核后才发布为正式照片

**2026-03-10**
- 上传时检测重复：若相同注册号+机型+日期已存在，给出提示

**2026-03-12**
- 整合：从统计页跳转画廊、细分机型搜索、重复上传提示 三项功能联调完成

**2026-03-13**
- 画廊照片卡片和筛选结果外部展示细分机型（`sub_model`）

**2026-03-14**
- manage 照片列表页新增搜索框，支持按注册号/机型/航司关键词筛选
- 审核队列支持 staff 在通过前直接修改提交信息（注册号、机型、航司等元数据）

**2026-03-15**
- manage 列表页筛选条件新增下拉选项，可按航司/机场/机型快速过滤
- 照片截图（screen photos）展示中加入细分机型显示

**2026-03-16**
- 以上多项功能（manage 搜索、审核编辑、筛选、细分机型显示）联调完成

**2026-03-18**
- 精选照片（Featured）区块改名
- 恢复随机照片（Random Photos）区块

**2026-03-19**
- 新增管理命令 `delete_entry`，支持通过命令行按 ID 批量删除 Photo 或 PendingPhoto 记录
- 首页汇总数据新增独立注册号计数（unique registrations）
- API 响应头注入 `X-Process-Time`（ms），前端性能面板可展示接口耗时

**2026-03-20**
- 全局搜索结果改为列表视图展示，信息密度更高

**2026-03-22**
- 首页航司排序改为按该用户上传照片数量排序
- 首页加载优化：开启 `GZipMiddleware`，拆分 airport 配置文件，减少首屏资源体积
- 新增 `ApiTimingMiddleware`，为所有 `/api/` 请求记录处理耗时并注入响应头

**2026-03-25**
- 画廊新增"只显示独立注册号"模式，每个注册号只展示一张（最新的）

**2026-03-28**
- 独立注册号模式下展示该注册号最近的拍摄日期
- 从统计页以航司+机型进入画廊时，自动展开注册号面板
- 新增管理命令 `clean_orphan_media`，查找并可选删除数据库中无引用的孤立媒体文件
- 待审核列表：已拒绝条目隐藏编辑区；已通过条目在操作区展示跳转到对应正式照片的链接（若照片已删除则提示）

**2026-03-29**
- 审核流程重构：审核通过/拒绝/删除照片时一律不删除物理文件，`PendingPhoto` 记录永久保留，文件清理统一由 `clean_orphan_media --delete` 处理
- `Photo` 删除时自动将关联的 `PendingPhoto.approved_photo` 置空，而非级联删除
- 新增 Staff 专用"替换图片"功能：直接上传新图覆盖，缩略图自动重新生成，旧文件交由孤立文件清理流程处理
